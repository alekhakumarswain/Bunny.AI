### 🔥 Why Bunny.AI?
**Bunny.AI** is a compact yet incredibly powerful autonomous agent. While other projects like OpenClaw are heavy and complex, Bunny.AI proves that **small lines of code can work better and faster.** 

Built on the **Gemini 2.5 Flash** backbone and equipped with a **Large Action Model (LAM)** philosophy, Bunny.AI doesn't just chat—it **Acts.**

### ✨ Key Features
- 🕒 **Time-Wise Autonomy**: Schedule tasks to run in the background while you sleep.
- 📧 **Active Email Pipeline**: Send reports, job applications, or notifications automatically.
- 🌐 **Dynamic Web Mastery**: Scrapes modern SPAs (React/Next.js) using its integrated **Playwright** engine—something standard scrapers fail at!
- 🔍 **Google-Powered Research**: Uses the official Google Search API for pinpoint accuracy.
- 💻 **Terminal Control**: Full access to your Windows system for file management and automation.

### 🚀 Getting Started

1.  **Clone & Install**:
    ```bash
    git clone https://github.com/alekhakumarswain/Bunny.AI.git
    cd Bunny.AI
    pip install -r requirements.txt
    playwright install chromium --with-deps
    ```

2.  **Configure `.env`**:
    Double-check your credentials in the `.env` file (see `.example.env`).

3.  **Run the Magic**:
    ```bash
    python app.py
    ```

### 🤝 Join the Development!
Bunny.AI is an open-source project and we would love your help to make it the most efficient LAM ever built! Whether you're fixing a bug, improving the README, or adding new features, your contribution is valuable.

#### How to Contribute:
1.  **Fork** the repository.
2.  **Clone** your fork to your local machine.
3.  **Create a New Branch** for your feature or fix.
4.  **Develop & Test**: Ensure Bunny's tools are still working smoothly.
5.  **Submit a Pull Request (PR)**: Explain clearly what you've added or changed.

#### 💡 Ways to Contribute:
- **New Skills**: Create new tools in the `Tools/` folder (e.g., WhatsApp automation, Database tools).
- **Optimization**: Help us find ways to keep Bunny's core code as small as possible.
- **Bug Fixes**: Help us tackle API 503 errors and scraper edge cases.
- **Documentation**: Improve this README or add wiki pages.

We review every Pull Request. Let's build the future of autonomous actions together! 🐰💪
